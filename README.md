# Dölken Lab Website
## Hannover Medical School (MHH)

This repository contains the source code for the Dölken Lab website, built using **R** and **Quarto**.

The website presents the lab's research, team members, publications, and related activities.

---

## Tech Stack

- R
- Quarto
- R Markdown
- HTML/CSS

---

## Local Development

### Requirements

- R (latest version recommended)
- Quarto (https://quarto.org)
- RStudio (recommended), but Positron also works.

### Preview Website Locally

```bash
quarto preview
```

Make sure you always work in the develop branch, or feature/something if it's very specific.
Then PR to main. Standard operating practices are a must.

---

## Editing a page - Crashcourse

1. Open the corresponding .qmd file (e.g. our_team.qmd).
2. Change what you want to change.
3. Save locally.
4. Render to verify it's working just fine.
5. Stage and push edits in develop or feature branch.
6. If everything looks good, ask for a PR to this repo.
